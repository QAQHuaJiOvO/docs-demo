### API

**2.0.0**及以上版本插件使用 init[配置各种属性](../customize/index.md)：

-   [init()](./init.md)

默认情况下，Fundebug 插件能够自动捕获未处理的错误(uncaught error)。另外，开发者也可以通过使用 Fundebug 提供的 API 发送其他错误信息:

-   [test()](./test.md)
-   [notify()](./notify.md)
-   [notifyError()](./notifyerror.md)
-   [leaveBreadcrumb()](./leavebreadcrumb.md)
