####监控 [GitBook](https://www.gitbook.com)

安装

```bash
npm install gitbook-plugin-fundebug
```

使用

```json
{
    "plugins": [
        "fundebug"
    ],
    "pluginsConfig":
    {
        "fundebug":
        {
            "apiKey": "API-KEY",
            "releaseStage": "development",
            "appVersion": "3.2.5"
        }
    }
}
```
